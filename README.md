## No AI Overview Script

This script adds '-ai' to every Google search query to remove the AI Overview.

### How to install

1. Download the extension Tampermonkey.
2. Click on the extension. Select 'Create a new script.'
3. Copy and paste the code from `script.js`.

### How to turn the script off

1. Click on the Tampermonkey extension. Select 'Dashboard.'
2. Toggle the script off.
